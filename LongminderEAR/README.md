----------

This application is divided into 5 subprojects:
LongminderEAR: EAR project to put it all together, and contains some sql scripts.
LongminderEJB: EJB project that handles back-end database handling using JPA, and email sending using JavaMail
LongminderEJBClient: Interfaces and data transfer objects for accessing LongminderEJB processes.
LongminderWAR: Dynamic web project that handles HTTP communication. Divided into JSF and REST portions.
JEE_utils: Non-project specific utilities
